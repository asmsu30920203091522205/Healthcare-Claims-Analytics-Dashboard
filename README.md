# Healthcare Claims Analytics Dashboard

## Project Overview

This project demonstrates the development of an interactive Healthcare Claims Analytics Dashboard using Power BI and MySQL.

The dashboard provides insights into healthcare insurance claims, claim status, hospital performance, insurance company analysis, and monthly claim trends.

---

## Tools Used

- Power BI
- MySQL
- SQL
- DAX
- Power Query
